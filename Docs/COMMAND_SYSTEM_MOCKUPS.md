## Design Principles

### Visual Hierarchy
1. **Categories** - Bold, with emoji and count
2. **Commands** - Clear label + description
3. **Positions** - Symbol + shares + P/L with color coding
4. **Actions** - Icon + clear action description

### Interaction Patterns
1. **Type to filter** - Instant search as you type
2. **Click to select** - Single click executes
3. **Hover for details** - Tooltip with more info
4. **Confirm destructive** - Always confirm closes/cancels
5. **Keyboard navigation** - Arrow keys + Enter

### Responsive Design
- **Desktop**: Full dropdown with all details
- **Tablet**: Compact view with essential info
- **Mobile**: Bottom sheet with touch-optimized buttons
